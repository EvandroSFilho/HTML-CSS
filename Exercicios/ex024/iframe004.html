Abaixo temos algumas ticas de segurançao para deixar o site mais seguro contra ataques

  • Usamos o sandbox="sandbox" para nao fazer pesquisas no site.
  • Usamos o sandbox="allow-same-origin" para liberar acesso apenas para sites com arquivos do mesmo arquivo.
  • Podemos usar tambem duas funcões no sandbox
  • Usamos o referrerpolicy="no-referrer" para nao deixar nenhum outro site pegar os dados para ser usado em outros fins.
